#   wav2krz examples/drumset.txt output.krz --mode drumset
#   wav2krz examples/drumset.txt output.for --mode drumset
#

# Without a root key, samples are assigned consecutive keys starting at C1.
# Key ranges (lo/hi) are optional — without them each sample occupies exactly one key.
